# Viva Aurora — Landing Page de exemplo

Landing de exemplo criada para demonstrar à Associação de Moradores do Bairro Aurora (projeto demo).

Arquivos:

- `index.html` — página principal
- `styles.css` — estilos e animações
- `script.js` — interações leves (IntersectionObserver + smooth scroll)

Imagens: URLs públicas do Unsplash usadas como exemplos (livre para demonstração). Antes de publicar, substitua por fotos oficiais da associação ou imagens licenciadas.

Como testar localmente:

1. Abra a pasta `c:\Dev\landin`.
2. Abra `index.html` no navegador (duplo-clique) ou sirva via um servidor local (recomendado):

```powershell
# no Windows (se tiver Python instalado)
python -m http.server 8000
# acessar http://localhost:8000
```

Deploy:

- Você pode usar o GitHub Pages: crie repo `landin`, envie os arquivos e ative Pages na branch `main` (pasta `/ (root)`).

Observações:

- Este é um exemplo rápido para apresentação ao cliente. Se o cliente aprovar, posso:
  - Substituir textos e imagens por conteúdo real
  - Adicionar formulário funcional (SMTP/Zoho/Forms)
  - Fazer deploy e configurar domínio
